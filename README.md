# NFT Metadata Generator

A professional utility for blockchain developers to automate the creation of NFT metadata. This script generates standardized JSON files compatible with OpenSea, Rarible, and other major marketplaces.

## Features
* **ERC-721 Compliant:** Generates metadata following the official Ethereum standards.
* **Trait Support:** Easily define attributes and properties for your digital assets.
* **IPFS Ready:** Structured to handle `ipfs://` CID prefixes for decentralized storage.

## Usage
1. Edit the `config.js` file to set your base URI and collection size.
2. Run the generator: `node generator.js`
3. Upload the resulting JSON files to IPFS.
